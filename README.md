# RouteGuard
RouteGuard is a full-stack authentication and authorization system built with MongoDB, Express, React, and Node.js featuring login, signup, logout, and role-based dashboard redirects. It uses React Context API to manage global authentication state and React Router v6 for protected routes, ensuring only authorized users can access specific pages. Users are redirected to different dashboards based on their roles (e.g., Buyer or Seller), while JWT tokens handle secure session management. This project demonstrates a clean implementation of role-based access control (RBAC), route guarding, and seamless frontend-backend integration, making it ideal as a starter template for MERN applications with authentication.
